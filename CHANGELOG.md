## 1.0.0

### Initial Release

**Directions (9)**
- topCenter, topLeft, topRight
- bottomCenter, bottomLeft, bottomRight
- leftCenter, rightCenter, center

**Animation Types (13)**
- slide, fade, scale, bounce, flip (3D X), flipY (3D Y), rotate
- elastic, zoom, ripple, drop, unfold, custom

**Card Styles (10)**
- glass (glassmorphism + backdrop blur)
- flat (white + coloured left border)
- gradient (animated shimmer)
- outlined (dark + glowing border)
- material (M3 elevation)
- minimal (pill shape)
- neon (pulsing neon glow)
- neumorphic (soft shadows)
- tinted (frosted coloured tint)
- custom (developer provides full widget)

**Dismiss Particle Effects (9)**
- none, burst, sparkle, confetti, bubbles
- firework, shatter, rippleBurst, hearts, custom

**Features**
- Toast queue — never overlaps
- Loading toast — never auto-dismisses
- Action buttons with dismissOnPress control
- Haptic feedback per type
- Animated countdown progress bar
- Pulsing icon animation
- Keyboard-aware positioning
- Swipe-to-dismiss (direction-aware)
- Global config via TwistConfig.setup()
- Full developer overrides: custom animation, custom effect, custom card
- Comprehensive widget test suite
